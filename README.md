# Simple Interest Calculator (Java)

A basic Java console program that calculates Simple Interest.

## Formula
Simple Interest = (P × R × T) / 100

- P → Principal  
- R → Rate of Interest  
- T → Time  

## How to Run

Compile:
javac Main.java

Run:
java Main

## Example

Enter principal (p): 1000
Enter rate of interest (r): 5
Enter time (t): 2
Simple Interest: 100.0

## Built With
- Java
- Scanner class
